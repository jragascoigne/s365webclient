# SENG365 Web Client

React web client for SENG365 2026 Assignment 2: a travel blogging client that talks to the Assignment 1 API.

## Setup

```sh
npm install
npm run dev
```

Set the API server with:

```sh
cp .env.example .env
```

Then edit `VITE_API_BASE_URL` if the reference API uses a different URL.

The assignment brief and working plan are in `docs/assignment-brief.md`.
